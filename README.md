# Core Java Projects

A collection of simple Core Java projects created for learning and practicing Java programming concepts. This repository contains beginner-friendly projects that help in understanding the fundamentals of Java through hands-on implementation.

## 📚 About

This repository is designed for students and beginners who want to strengthen their Core Java skills by building small projects. Each project focuses on different Java concepts such as:

* Variables and Data Types
* Conditional Statements
* Loops
* Arrays
* Methods
* Object-Oriented Programming (OOP)
* Classes and Objects
* Inheritance
* Polymorphism
* Encapsulation
* Abstraction
* Exception Handling
* Collections Framework
* File Handling

## 🚀 Projects Included

Some example projects in this repository:

* ATM Management System
* Number Guessing Game
* Student Management System
* Banking System
* Calculator
* Library Management System
* Employee Management System
* Quiz Application
* Simple Login System
* Other Java Practice Programs

## 🛠️ Technologies Used

* Java
* JDK 8+
* VS Code / IntelliJ IDEA / Eclipse

## 🎯 Purpose

The main goal of this repository is to:

* Practice Core Java concepts
* Improve problem-solving skills
* Build a strong foundation for Java development
* Prepare for coding interviews and placements
